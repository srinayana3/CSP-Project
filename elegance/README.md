# CSP-Project-ELEGANCE
Elegance is a modern interior design website built using the MERN Stack.

# Git hub link
https://github.com/srinayana3/CSP-Project

# The project consists of five main pages:
•	Home\
•	About\
•	Services\
•	Portfolio\
•	Contact

# Tech Stack
Frontend: React.js,Vite\
Backend: Node.js, Express.js\
Database: MongoDB

# How to Run

Frontend:\
npm install\
npm run dev

 Backend:\
cd backend\
npm install express cors mongoose dotenv nodemon\
node server.js  //  or npx nodemon server.js
